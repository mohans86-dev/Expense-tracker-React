# 💰 Expense Tracker App (React)

A modern and responsive Expense Tracker Web Application built using **React.js**.  
This application allows users to manage their expenses, add balance, edit transactions, and track their real-time financial status.

---

## For Demo
Visit the link: https://expense-tracker-app-ten-vert.vercel.app/

---

## 🚀 Features

- ✅ Add new expense transactions
- ✅ Add balance to main account
- ✅ Edit existing transactions
- ✅ Delete individual transactions
- ✅ Clear all transactions
- ✅ Real-time balance calculation
- ✅ Default balance initialization (₹10,000)
- ✅ Responsive and modern dark-themed UI

---

## 🛠️ Tech Stack

- **Frontend:** React.js (Functional Components + Hooks)
- **State Management:** useState
- **Styling:** SCSS (Nested styling)
- **Unique ID Generation:** UUID
- **Build Tool:** Vite

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/your-username/your-repo-name.git
```

2. Navigate into the project folder

```
cd react-expense-tracker
```

3. Install dependencies

```
npm install
```

4. Run the development server

```
npm run dev
```

---

## 📸 Application Overview

- Users can add expenses with name and amount.
- Users can add balance directly to the main balance.
- The balance updates automatically based on transactions.
- Transactions are displayed with type-based styling (Income / Expense).
- All data is managed using React state.
